# new-genotype-explorer
주제 융합 탐구 프로젝트에서 생명과학과 컴퓨터공학을 합한 사이트를 만들었다. "표현형과 가족 정보를 이용하여 가능한 유전자형을 얼마나 정확하게 좁혀 나갈 수 있을까?"  사람은 자신의 표현형(우성/열성)은 쉽게 알 수 있지만, 대부분 자신의 정확한 유전자형(AA, Aa, aa)은 알 수 없다.  이 프로젝트에서는 부모, 자녀, 조부모의 표현형을 이용하여 가능한 부모 유전자형을 모두 탐색하고, 입력되는 정보가 많아질수록 조건에 맞지 않는 경우를 제거하는 알고리즘을 구현한다.  이를 통해 생명과학의 유전 원리와 컴퓨터공학의 탐색 알고리즘을 융합한다.
